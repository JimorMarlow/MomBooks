# Файловый сервер для старого iPad

У мамы самый первый iPad, на который уже нельзя устанавливать программы. Safari устарел и не может открывать современные файлы для получения почты. Мне нужно закачивать ей книги в формате epub.

## HFS

Для загружки книг используется базовый HFS (Http File Server) без секьрности с простым паролем. Он дает возможность в локальной сети подключиться к папке /hfs и показать все содержимое папки в виде ссылок. На файлы с расширением epub мама будет нажимать и загружать себе книги в iBooks. Я буду сам скачивать книги и раскладывать по папкам и авторам.

## Требования

Нужно создать скрипт на Python update_books_index.py, который будет сканировать все содержимое папке /hfs и создавать файл index.html как начальную страницу http file server. Нужно построить таблицу в виде книжной полки:
папка: книга1 книга2 ...

в самом верху полоса "Новое:" где показаны новые книги за последние 2 недели.
Ниже папки по алфавиту.
В самом низу полоса с книгами в корневой папке.

В индекс попадают только файлы с расширением epub.

Скрипт должен сделать обложку книги, снизу подпись: автор и название. Если удасться найти обложку в интернете, то положить в папку hfs/images и дать ссылку на нее. Если обложки нет, то сгенерировать изображение книги: сверху автор, по центру крупно название, снизу год. Данные взять из epub. Сгенерированные обложки класть в ту же папку hfd/images, имена обложек пуст совпадают с именами файла ebub.

## Использование

После загрузки новых книг, я буду запускать этот скрипт. Для новых книг, для которых нет обложки нужно загрузить или создать обложку и обновить индексный файл.

- Заведи файл hystory.md и записывай все, что делаешь, чтобы можно было потом продолжить в следующем сеансе.
