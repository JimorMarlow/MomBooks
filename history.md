# История разработки MomBooks

## 2026-03-11

### Начат проект
- Прочитано ТЗ из docs/mombooks.md
- Изучены правила из docs/qwen_make_no_mistakes.md
- Установлена библиотека ebooklib для работы с epub

### Исследование структуры epub
- Изучена структура epub файлов в hfs/books
- Выяснено, что:
  - Метаданные хранятся в DC namespace (http://purl.org/dc/elements/1.1/)
  - Creator содержит автора
  - Title содержит название
  - Date содержит год издания
  - Обложка может быть встроена как image/jpeg (images/cover.jpg)
  - Некоторые файлы конвертированы из FB2 и имеют специфичные метаданные

### Создан скрипт update_books_index.py
Функционал:
- Сканирование всех .epub файлов в hfs/books (рекурсивно)
- Извлечение метаданных: автор, название, год
- Извлечение встроенной обложки из epub
- Генерация обложек для книг без обложек (текст на цветном фоне)
- Сохранение обложек в hfs/images (имя = имя epub + .jpg)
- Генерация index.html с книжной полкой

### Структура index.html
- Секция "Новое" — книги за последние 2 недели
- Папки по алфавиту
- Книги в корневой папке (внизу)

### Результат тестирования
- Найдено 59 книг в 6 папках
- Для некоторых книг обложки извлечены из epub
- Для остальных сгенерированы обложки с текстом
- index.html создан успешно

### Как использовать
После добавления новых книг запустить:
```
python update_books_index.py
```

### 2026-03-11 (Исправление)
**Проблема:** iPad не мог открыть файлы в папках с кириллическими именами (например, "Гарри Поттер")

**Решение:** Добавлено URL-кодирование для путей к файлам и обложкам
- Кириллица кодируется как %D0%xx
- Пробелы кодируются как %20

Пример: `Гарри Поттер/книга.epub` → `%D0%93%D0%B0%D1%80%D1%80%D0%B8%20%D0%9F%D0%BE%D1%82%D1%82%D0%B5%D1%80/%D0%BA%D0%BD%D0%B8%D0%B3%D0%B0.epub`

### 2026-03-11 (Исправление интервалов для старого Safari)
**Проблема:** На старом iPad Safari книги отображались вплотную друг к другу без интервалов

**Причина:** Старый Safari не поддерживает CSS-свойство `gap` для Flexbox

**Решение:**
- Добавлен `margin: 10px` для `.book-card` — явные отступы между книгами
- Добавлен `margin: -10px` для `.books-row` — компенсация внешних отступов по краям контейнера

**Изменённые файлы:**
- `update_books_index.py` — обновлены CSS-стили в генераторе HTML
- `hfs/index.html` — перегенерирован с новыми стилями
