# MomBooks — Домашняя библиотека для iPad

Файловый сервер для загрузки книг на старый iPad через локальную сеть.

## Структура проекта

```
MomBooks/
├── update_books_index.py    # Скрипт сканирования и генерации index.html
├── docs/                     # Документация
├── hfs/
│   ├── books/               # Книги в формате epub (не коммитится)
│   ├── images/              # Обложки книг (не коммитится)
│   └── index.html           # Главная страница для HFS (не коммитится)
└── history.md               # История разработки
```

## Использование

1. Установите зависимости:
   ```bash
   pip install ebooklib pillow
   ```

2. Добавьте книги в папку `hfs/books/` (можно создавать подпапки по авторам/сериям)

3. Запустите скрипт для генерации index.html:
   ```bash
   python update_books_index.py
   ```

4. Запустите HFS (Http File Server) с корневой папкой `hfs/`

5. Откройте `http://<ваш-ip>/index.html` на iPad

## Требования

- Python 3.8+
- ebooklib
- Pillow

## Примечания

- Книги и обложки не коммитятся в репозиторий (см. `.gitignore`)
- Для книг без обложек скрипт автоматически генерирует обложки с текстом
- Новые книги (за последние 2 недели) отображаются в верхней полосе "Новое"
